# Sample contract sources — PUBLIC DATA ONLY

Everything in this repo is shareable publicly. No private or client data is ever
committed. Sample contracts come exclusively from:

- **CUAD** (Contract Understanding Atticus Dataset) — 510 commercial contracts,
  CC BY 4.0. https://www.atticusprojectai.org/cuad
- **SEC EDGAR** — public company filings (material agreements are public record).
  https://www.sec.gov/cgi-bin/srqsb?text=edgar

`scripts/fetch_samples.py` downloads a small, fixed set of these for local dev and
for the eval corpus. License/attribution for each file is recorded here when added.
